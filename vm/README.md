# VM Build Example

## Prerequisites

We need arp to be installed

Debian

    apt-get install net-tools

Ubuntu

    apt-get install net-tools

Alpin

    apk add net-tools
    
Arch Linux

    pacman -S net-tools
    
Kali Linux

    apt-get install net-tools
    
CentOS

    yum install net-tools
    
Fedora

    dnf install net-tools
    
Raspbian    
    
    apt-get install net-tools
    
Docker

    docker run cmd.cat/arp arp
